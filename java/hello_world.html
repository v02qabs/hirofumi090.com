Java で helloworld


Java で、 hello world をコンソールに 表示するまで。 ソースコード：　hello_word.java 
import java.io.*;

public class hello world{
	public static void main(String[] args){
		System.out.println("Hello");
	}
}
コンパイル
#javac hello_world.java

実行：
#java hello_world
